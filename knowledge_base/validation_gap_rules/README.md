# Validation Gap Rules

## Purpose

This directory stores structured rules for identifying common analytical method validation gaps.

Rules should be aligned with the platform's evidence labeling principles and should not overclaim regulatory certainty.

## Rule Categories

- Specificity / selectivity gaps
- Accuracy design gaps
- Precision design gaps
- Linearity / calibration model gaps
- Range / reportable range gaps
- LOD / LOQ justification gaps
- Robustness gaps
- System suitability gaps
- Sample / reagent stability gaps
- Method transfer gaps
- Bioassay-specific gaps
- Multivariate / chemometric method gaps

## Expected Rule Fields

- rule id
- method type
- validation characteristic
- trigger condition
- gap severity
- rationale
- suggested reviewer question
- evidence level
- human review requirement
