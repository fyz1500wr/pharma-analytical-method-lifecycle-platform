# Analytical Technology Registry

## Purpose

This directory stores entries for established, complex, and emerging analytical technologies.

The registry supports the Technical Context-Aware Analytical Report Review Module.

## Technology Status Categories

```text
Technology Status:
- Known and internally mapped
- Known but not internally mapped
- Emerging / insufficient internal knowledge
- Unclear terminology / requires clarification
```

## Example Technologies

- HPLC / UPLC
- SEC-HPLC
- CE-SDS
- icIEF
- LC-MS / LC-MS/MS
- Multi-attribute method (MAM)
- ELISA
- qPCR / ddPCR
- Cell-based potency assay
- Reporter gene assay
- SPR / BLI binding assay
- NIR / Raman / PAT / chemometric methods
- NGS-based analytical methods

## Expected Entry Fields

- technology name
- aliases
- principle summary
- typical analytical purpose
- applicable product types
- key controls
- common failure modes
- common validation concerns
- regulatory considerations
- literature context
- SME review triggers
- evidence level
