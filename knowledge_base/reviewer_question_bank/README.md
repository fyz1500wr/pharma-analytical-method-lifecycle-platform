# Reviewer Question Bank

## Purpose

This directory stores reusable questions for reviewing analytical method development reports, validation protocols, validation reports, transfer reports, and vendor technical proposals.

## Question Types

- Clarification question
- Critical gap question
- Major gap question
- Minor gap question
- SME escalation question
- Vendor / lab follow-up question
- Regulatory impact question

## Example Questions

- Please clarify the intended analytical purpose of this method and whether it is intended for release, stability, characterization, comparability, or transfer.
- Please justify how the selected reportable range covers the expected sample concentration and specification range.
- Please provide data demonstrating that the system suitability criteria monitor the critical performance of the method.
- Please clarify whether matrix interference was evaluated for the actual drug product matrix.
- Please provide justification for acceptance criteria and whether they are based on development data, prior knowledge, or regulatory expectation.

## Feedback Loop

Frequently used or high-value reviewer questions should be promoted into method-specific checklists.
