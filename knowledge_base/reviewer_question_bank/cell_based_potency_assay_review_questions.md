# Cell-Based Potency Assay Review Questions

## Purpose

This file provides reusable reviewer questions for cell-based potency assay reports, validation protocols, validation reports, method transfer reports, and vendor technical proposals.

These questions are intended for draft review support only and require human judgment before use.

---

## 1. Intended Purpose

1. Is the method intended for development, characterization, release, stability, comparability, method transfer, or regulatory filing support?
2. Does the report clearly state the intended purpose and reportable result?
3. Does the data package support the intended use, or does it only support feasibility?

---

## 2. Biological Relevance

1. What biological response does the assay measure?
2. How is the response linked to the product mechanism or intended potency attribute?
3. Is the assay response specific enough to support the claimed potency conclusion?
4. Is an orthogonal method needed to support the conclusion?

---

## 3. Reference Standard and Controls

1. What reference standard or working standard is used?
2. Is the standard qualified for the intended use?
3. Are storage, handling, preparation, and stability of the standard described?
4. Are positive, negative, matrix, and cell controls included?
5. Are control acceptance criteria justified?

---

## 4. Matrix Effect and Parallelism

1. Was the actual product matrix evaluated?
2. Was dilution linearity or dilution-response comparability assessed?
3. Was parallelism between reference standard and sample assessed where applicable?
4. If matrix effect or parallelism was not assessed, what is the justification?
5. Does the sample preparation or dilution scheme reduce or mask matrix interference?

---

## 5. Curve Model and Reportable Range

1. What curve model is used?
2. Is the model appropriate for the assay response?
3. Is weighting used, and if so, why?
4. Does the range cover expected sample concentration and specification range?
5. Are upper and lower response limits defined?
6. Are out-of-range samples handled by a predefined rule?

---

## 6. System Suitability and Run Validity

1. What criteria determine whether a run is valid?
2. Do system suitability criteria monitor critical assay performance?
3. Are assay window, control response, curve fit, and replicate variability addressed?
4. Are failed-run or repeated-run rules predefined?
5. Are excluded runs documented and justified?

---

## 7. Precision and Robustness

1. Are repeatability and intermediate precision assessed?
2. Are analysts, days, instruments, reagent lots, and cell passages considered where relevant?
3. Is assay variability acceptable for the intended use?
4. Are critical parameters identified?
5. Is robustness evaluated or planned?

---

## 8. Stability Use

1. If the method supports stability testing, can it detect potency loss or biologically meaningful change?
2. Are stressed or degraded samples evaluated where appropriate?
3. Does the report distinguish content loss from activity loss?
4. Are stability trends interpretable given assay variability?

---

## 9. SME Review Triggers

Escalate to SME review when:

- release or stability decisions are supported;
- matrix effect is unclear;
- parallelism is missing;
- reference standard qualification is unclear;
- curve fitting or statistical model is complex;
- assay variability is high;
- external lab report lacks raw data or failed-run rationale;
- report conclusion exceeds supporting data.
