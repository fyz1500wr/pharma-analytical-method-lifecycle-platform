# Evidence Packs

## Purpose

This directory will store normalized and version-controlled evidence packs used by Claude, Codex, MCP servers, workflows, and dashboards.

Evidence packs may include:

- official regulatory source metadata
- guidance summaries
- applicability mappings
- retrieval timestamps
- source URLs
- version / effective date information
- human review status
- affected workflows or templates

## Important Rule

Evidence packs should be source-traceable. Do not store unverified AI summaries as authoritative evidence without labeling them as AI-assisted and requiring human review.
