# Common Analytical Method Failure Modes

## Purpose

This directory stores recurring technical and documentation gaps observed during analytical method development, validation, transfer, and laboratory report review.

## Example Failure Modes

- Method purpose not aligned with CQA
- Reportable range not covering specification or sample concentration
- Specificity not demonstrated for matrix or degradation products
- Stability-indicating claim not supported by forced degradation data
- Bioassay lacks parallelism or matrix interference assessment
- System suitability does not monitor the critical method performance
- Reference standard or working standard control is insufficient
- Accuracy / precision design does not match intended use
- Method transfer lacks comparative acceptance criteria
- External lab conclusion not supported by raw data or plots

## Feedback Loop

When a gap is found during review, it should be captured here or in a method-specific checklist so future protocol generation and report review can automatically flag it.
