# Phase 1 MVP Scope

## Scope Statement

Phase 1 will build the first usable product-to-method recommendation capability.

Initial MVP scope:

```text
Biologics / recombinant proteins / drug product / release and stability
```

## In Scope

- Product profile intake fields
- Product-method matrix expansion
- Required / conditional / optional analytical category classification
- Method type suggestion
- Missing information list
- Risk flags
- Evidence level labeling
- Human review statement

## Out of Scope for Phase 1

- Full regulatory evidence pack automation
- Full MCP implementation
- Full dashboard
- Final validation protocol generator
- Automated approval of analytical strategy

## Success Criteria

Phase 1 is successful when the repository can produce a structured method selection output from a synthetic product profile and clearly label rationale, evidence level, missing information, and human review requirement.
