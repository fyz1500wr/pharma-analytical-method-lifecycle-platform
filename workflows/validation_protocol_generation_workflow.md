# Validation Protocol Generation Workflow

## Purpose

Generate an analytical method validation protocol draft based on method purpose, product profile, method type, and applicable validation expectations.

## Inputs

- Product profile
- Analytical method profile
- Validation scope
- Intended purpose
- Sample matrix
- Method type
- Source priority matrix
- Applicable internal rules

## Steps

1. Confirm intended purpose and reportable result.
2. Confirm validation scope: full validation, partial validation, method transfer, co-validation, verification, or other.
3. Select validation characteristics.
4. Propose experimental design.
5. Propose system suitability tests.
6. Propose sample, standard, and control strategy.
7. Draft acceptance criteria rationale.
8. Add deviation handling principles.
9. Add evidence labels and human review statement.

## Outputs

- Validation protocol outline
- Validation characteristic matrix
- Experimental design summary
- Acceptance criteria draft
- System suitability proposal
- Deviation handling rule
- Human review statement
