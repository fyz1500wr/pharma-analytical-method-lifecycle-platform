# Sample Data Governance

## Purpose

This project may use sample input and output files for testing workflows. Sample data must be non-confidential and clearly marked as synthetic.

## Rules

1. Do not upload confidential product records.
2. Do not upload raw laboratory reports without sanitization.
3. Do not upload vendor documents unless approved for repository use.
4. Do not include patient data, personal data, or controlled documents.
5. Examples should be generic and clearly labeled as synthetic.
6. Any GxP-relevant example must include a human review statement.

## Planned Synthetic Examples

- Generic product profile example
- Generic method selection output
- Generic validation protocol draft
- Generic technical context card
- Generic lab report gap assessment
